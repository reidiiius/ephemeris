#### Usage
command line

    cd ephemeris | && chromium --incognito index.html

optional browsers

    firefox index.html

    epiphany index.html

    midori index.html

    dwb -S index.html

    dillo -f index.html

text-based browsers

    elinks index.html

    cat index.html | w3m -T text/html
