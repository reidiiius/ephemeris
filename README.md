#### Usage
command line

    cd ephemeris && iceweasel index.html

optional browsers

    chromium --incognito index.html

    epiphany index.html

    dwb -S index.html

    dillo -f index.html

text-based browsers

    elinks index.html

    cat index.html | w3m -T text/html
